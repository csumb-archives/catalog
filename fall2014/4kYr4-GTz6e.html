In order to graduate, you will also need to complete your general education and university requirements.

Complete ALL of the following courses: 
  • MATH 170: Discrete Mathematics
  • MATH 150: Calculus I 
  • CST 231: Problm-Solving/Programng
  • CST 238: Intro to Data Structures
  • CST 201: Media Tools
  • CST 300: Major ProSeminar
  • CST 337: Computer Architecture
  • CST 373: Ethics in Comm & Tech
  • CST 400: Capstone Project Planning
  • CST 401: Capstone Lab
  • CST 496: Directed Study taken twice, once concurrently with CST 400: Capstone Project Planning and once concurrently with CST 401: Capstone Lab
  • MATH 151: Calculus II OR MATH 370: Logic & Computation
  • CST 462S: Computer Sci and Community OR CST 361S: Technology Tutors

Complete ONE of the following concentrations:

Network & Security Concentration
Complete ALL of the following courses:
  • CST 311: Intro to Computer Networks
  • CST 312: Network Security
  • CST 412: Network Administration
Information Systems Concentration
Complete ALL of the following courses:
  • CST 363: Intro to Database Systems
  • CST 365: Business & Information Tech
  • CST 336: Internet Programming OR CST 463: Database System Administration
Softare Engineering Concentration
Complete ALL of the following courses:
  • CST 338: Software Design
  • CST 438: Software Engineering
  • CST 336: Internet Programming
Game Development Concentration
Complete ALL of the following courses:
  • CST 306: Game Engine Programming
  • CST 320: Intro to Digital Games
  • CST 438: Software Engineering
All students must complete THREE CSIT elective courses from the list below and not used to fulfill their concentration:
  • CST 306: Game Engine Programming
  • CST 311: Intro to Computer Networks
  • CST 312: Network Security
  • CST 320: Intro to Digital Games
  • CST 334: Operating Systems
  • CST 336: Internet Programming
  • CST 338: Software Design
  • CST 363: Intro to Database Systems
  • CST 365: Business & Information Tech
  • CST 370: Design&Analysis of Algorithms
  • CST 412: Network Administration
  • CST 394: Internship
  • CST 438: Software Engineering
  • CST 463: Database System Administration
  • CST 495: Special Topics
  • CST 496: Directed Study
