> Graduate Degrees

Graduate Degree Programs

CSUMB offers four master’s degree programs, one of which has two distinct options. To view descriptions of the programs, select from the list in the left navigation.

Learning Outcomes

To focus on outcomes, each of the CSUMB graduate programs requires students to display achievement of a set of Learning Outcomes (MLOs) established by the program faculty. The description of each program includes a list of LOs and associated coursework.
