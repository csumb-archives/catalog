Course Description Management Directions

Important Notes

Though you edit course descriptions in Charlotte, that page will never be publicly linked. Instead, at the end of workflow, Marketing & Publications copies and pastes any changed descriptions into Banner. Automated scripts query the Banner information daily and deliver the results to the Catalog and Schedule websites, as well as other online locations.

Because the title, credit, prereq, coreq, enrollment restriction, instructor consent, and other fields are maintained in Banner, that information does not appear on this Charlotte course description page. Only the course number and description body appear here, with the former serving as an alphanumeric reference ensuring that the description gets pasted into the appropriate Banner location.

To Revise an Existing Course Description
  • Delete and type in text as appropriate within the current description. Please, do not cut the whole description and paste in a new one.
  • Check to ensure that you have followed all guidelines and standards. Note that descriptions exceeding the 500-character limit will be truncated when pasted to Banner.
  • Enter the current date in place of the previous date at the end of the Last updated in Charlotte line that appears between the course number and description. Do not change the date at the end of the Last updated in Banner line. The former date being later than the latter triggers M&P to update the description in Banner.
To Add the Description for a New Course
  • Find the appropriate alphanumeric position on this page.
  • Add a line, then copy and paste the following boilerplate block onto that line:
PREF###
  • Last updated in Charlotte:
  • Last posted to Banner: Never

 Description

  • Replace the PREF### field with the correct prefix and number.
  • Replace the Description field by typing or pasting in your description of the course. If pasting, you must scrub all code off the text as guided by the Pasting External Text tutorial on the Charlotte website.
  • Check to ensure that you have followed all guidelines and standards. Note that descriptions exceeding the 500-character limit will be truncated when pasted to Banner.
  • Enter the current date at the end of the Last updated in Charlotte line that appears between the course number and description. Do not enter a date at the end of the Last updated in Banner line. The absence of a date in the latter triggers M&P to add the description in Banner.
