You can use any of the following methods to find the information you need on this CSUMB.EDU/catalog website:

  • Select the section you want in the navigation, then choose from the second, third and fourth navigation levels to find the exact page you need.
  • Enter keywords in the "Search this site" box below the navigation on any catalog page, then hit return and choose from the results.

If you don't find what you need, please use the Questions/Feedback link to let us know. Also, we constantly strive to improve both the information in the catalog and the tools provided to help you navigate successfully to what you need, we also invite you to use this link to send us your suggestions.
