CMS-HR Self-Service

All state and Foundation employees (excluding student assistants) are able to update certain information on-line using PeopleSoft Human Resources: https://cmsweb.csumb.edu/servlets/iclientservlet/HMBPRD/

The items that can be updated are:

  • Home Address
  • Emergency Contact Information
  • Preferred Name (Please note that Legal name must still be changed in writing using an Employee Action Request with appropriate documentation.)
  • Home Telephone Number

There are job aids to assist you in completing these updates. If you did not receive an email with your User ID and password, please send an email to CMS DBA. If you've forgotten your password and did not set up the forgotten password link, please send an email to CMS Questions.
