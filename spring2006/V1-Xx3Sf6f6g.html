CSUMB Library ~ Current & Planned

The current CSUMB Library provides full services from Building 12 on the Central Campus Quad. It also provides extensive online services on the current CSUMB Library website.

A new library will be completed in 2008. The building project and fundraising campaign are detailed on the Tanimura & Antle Family Memorial Library webpages.

To improve services now and prepare effectively for the new facility, the CSUMB Library is currently conducting a user survey. Please participate. You might win an iPod!
