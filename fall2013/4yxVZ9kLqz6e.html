*Guidelines for Summer and Alternative Sessions add/drop period:

  • Classes less than four weeks must be added/dropped prior to the start of the class
  • Classes four/five weeks long must be added/dropped within the first two days of the class 
  • Classes eight/ten weeks long must be added/dropped within the first three days of the class
